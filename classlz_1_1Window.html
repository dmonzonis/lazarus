Detailed Description

Handles the main window of the game.

The window is what will be shown to the user, and where all the sprites are rendered.

Coordinates in the window are given in tiles, where (0, 0) is the top-left corner, with the x-axis increasing to the right and the y-axis increasing when going down.

Draws the tile with the given ID from the tileset at the given position, in window coordinates.

If a tile was already drawn in the given position before flushing the window, it will be drawn on top.
